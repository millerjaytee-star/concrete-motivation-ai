# Deploy Concrete Motivation Website Now

## Fastest Path: Netlify

1. Go to Netlify.
2. Choose **Add new site**.
3. Choose **Import an existing project**.
4. Connect GitHub.
5. Select this repo:

```text
millerjaytee-star/concrete-motivation-ai
```

6. Use these settings:

```text
Build command: leave blank
Publish directory: website
```

7. Deploy site.
