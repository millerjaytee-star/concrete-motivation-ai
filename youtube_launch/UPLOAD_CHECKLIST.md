# Upload Checklist

Use this checklist before every YouTube upload.

## Video Details

- [ ] Title is clear, direct, and searchable.
- [ ] Description includes the main idea, audience, challenge, website link, and subscribe CTA.
- [ ] Thumbnail uses black/concrete/gold style and 2-4 bold words.
- [ ] Tags/keywords include Concrete Motivation, Jaytee Miller, motivation, discipline, leadership, fatherhood, athletes, mindset, podcast, and the specific topic.
- [ ] Chapters are added for long-form videos.
- [ ] Pinned comment includes the viewer challenge and website link.
- [ ] Shorts cutdown is planned.
- [ ] Community post is drafted.
- [ ] Website link is included.
- [ ] Booking CTA is included for speaking, workshops, podcast, or leadership inquiries.

## Quality Check

- [ ] Audio is clear.
- [ ] First 5 seconds create a reason to keep watching.
- [ ] Message is original Concrete Motivation voice.
- [ ] No copied public speaker language.
- [ ] No fake luxury claims.
- [ ] No unsupported claims or unverified scripture quotes.
- [ ] CTA asks for one clear action.

## Output Vault Check

- [ ] Script or outline saved to the output vault.
- [ ] Description saved for reuse.
- [ ] Pinned comment saved.
- [ ] Thumbnail text saved.
- [ ] Shorts cutdown notes saved.

## Post-Publish

- [ ] Pin the comment.
- [ ] Add the video to the correct playlist.
- [ ] Share the link on the website when ready.
- [ ] Create 1-3 Shorts from the strongest moments.
- [ ] Post a Community question.
- [ ] Capture viewer questions for future Concrete Conversations topics.
