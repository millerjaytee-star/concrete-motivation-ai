# Social Handoff

Use this checklist before any public post.

## Required Review

- Confirm the YouTube video is private or explicitly approved for public release.
- Confirm no private lead data, API keys, OAuth tokens, or customer details appear in captions or screenshots.
- Confirm the caption uses the Concrete Motivation voice: direct, disciplined, real, family-centered, faith-aware, and practical.
- Confirm the post points back to the website or booking path when appropriate.

## Launch Asset Queue

1. YouTube private verification link
2. 30-second Short caption
3. Instagram Reel caption
4. LinkedIn founder note
5. Community post question
6. Gmail draft for warm contacts

## Manual Posting Rule

Automation may prepare copy and checklists. A human must approve and perform public posting unless a future issue explicitly adds a governed scheduler.
