# Concrete Motivation Operating System

Concrete Motivation runs as an offline-first operating system for content, outreach, CRM, podcasting, and executive review.

## Core Layers

- Command center menu in `main.py`
- Specialist bots in `concrete_motivation/`
- Content factory outputs in `outputs/content_packages/`
- YouTube packages in `outputs/youtube_packages/`
- Outreach campaigns in `outputs/outreach_messages/`, `outputs/sales_outreach/`, and `outputs/podcast_production/`
- CRM master pipeline in `outputs/crm/concrete_motivation_pipeline.csv`
- Executive dashboard in `dashboard/ceo_dashboard.py`

## Daily Use

1. Open the command center.
2. Check the executive dashboard.
3. Build content or outreach assets.
4. Add leads to CRM.
5. Review follow-ups and publish only after manual review.
