# Website Revenue Funnel

The revenue website pages live in `website_content/` and are safe to generate locally.

## Pages
- `membership.md`
- `speaking.md`
- `sponsors.md`
- `book_jaytee.md`
- `payment_links.md`

## Required sections
- Headline
- Problem
- Promise
- Offer
- Who it is for
- Benefits
- CTA button text
- Payment link placeholder
- Booking/contact fallback

## Run it
```bash
python3 scripts/update_revenue_website_pages.py
```
