# Membership Program

Concrete Builders Membership is the recurring offer for Concrete Motivation.

## Included assets
- Membership sales page
- Monthly offer
- Annual offer
- FAQ
- Terms placeholder
- Welcome email
- Renewal / nurture email
- Cancellation-save email
- Member onboarding checklist
- Weekly challenge template

## Run it
```bash
python3 scripts/create_membership_program.py --monthly-price 40 --annual-price 400
```
