# Concrete Motivation Website

Version 6 adds a static website foundation in `website/` for Concrete Motivation and Concrete Conversations.

## Preview locally

Open `website/index.html` in a browser. No backend, build step, API key, or deployment is required.

## Sections

- Hero with Concrete Motivation brand, tagline, and calls to action
- Founder story
- Speaking topics
- Concrete Conversations podcast
- Programs and offers
- Early partner testimonial placeholders
- Booking/contact placeholder form
- Footer with social link placeholders

## Design direction

The site uses a black, concrete gray, white, and gold palette with a generated hero image at `website/assets/concrete-hero.png`. The page is mobile-first, semantic, and built for schools, teams, churches, businesses, and athletes.

## Notes

The booking form is static. It prevents page reload and shows a placeholder message, but it does not send data until a form service is connected. The hero image is stored at `website/assets/concrete-hero.png`.
