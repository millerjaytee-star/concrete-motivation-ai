# Daily Execution Workflow

Use this workflow to keep Concrete Motivation moving every day.

## Morning

1. Run the executive dashboard.
2. Review pending follow-ups.
3. Choose the priority for the day.

## Midday

1. Build one content package or content batch.
2. Generate one outreach campaign if leads need attention.
3. Add or update CRM leads.

## Afternoon

1. Review any YouTube package or dry run.
2. Tighten copy before publishing.
3. Save the best output to the local vault.

## End of Day

1. Check the CRM dashboard.
2. Confirm next follow-up dates.
3. Pick tomorrow's one highest-value action.
